# javafx-desktop-apps
## JavaFx Password Manager
### OVERVIEW
Password Manager App allows to display user's passwords in table form.
### FEATURES
1. Display user's password.
2. Add new password.
3. Copy login or password (when table cell is selected).
4. Search by title.
